Pi Served: Installing Raspbian Jessie Lite

Installing Raspbian Jessie Lite is a bit of a misnomer because you’re really not installing it, but rather copying a pre-installed image to a Micro-SD card.

The following method should allow you to install and use your Pi in a completely headless mode (i.e., no need to connect a keyboard, mouse, or monitor directly to the Pi)

This guide assumes you are using a Mac to transfer the image to the Micro-SD card. While the steps should be similar on Linux, I recommend you skip my directions and go straight to the provided references if you’re using Windows.

  1. Go to your home directory and download the latest Raspbian Jessie Lite image from the Raspberry Pi download page:

    cd ~/ && curl -L https://downloads.raspberrypi.org/raspbian_lite_latest -o raspbian-jessie-lite.zip

  2. Verify the SHA-1 Sum of the downloaded file to what’s published on the Raspberry Pi download page:

    shasum raspbian-jessie-lite.zip

  3. Unzip the download image (this may take a minute or so depending on your hardware):

    unzip raspbian-jessie-lite.zip

  4. Open Terminal and type the following command without your Micro-SD inserted in your computer:

    df -h

    The above command will list all of your connected disk devices (and associated capacity information). Any devices (/dev/disk#) in this list must not be used in the subsequent dd commands. This cautionary step to help you avoid losing important data.

  5. Insert the Micro-SD card into your computer and re-run the display free disk space command you used above and make note of the device reference which did not exist in the prior step:

    df -h

    Although the remaining steps will use /dev/disk2, you must update each disk reference below to match the new disk number reported by your computer.

  6. It’s now time to unmount (not eject or remove) the Micro-SD card so you can write the Raspbian image to it in the next step:

    sudo diskutil unmount /dev/disk2s1

  7. With the card unmounted, it’s time to actually transfer the image file (revise the filename to match file you unzipped earlier):

    sudo dd bs=1m if=2015-11-21-raspbian-jessie-lite.img of=/dev/rdisk2

    Be patient as this will take several minutes.

  8. After the above step completes successfully, eject the card:

    sudo diskutil eject /dev/rdisk2

  9. Now it’s time to insert the card into the Pi and connect to it over SSH (default password is “raspberry”):

    ssh pi@192.168.1.2

    Change the above address to match the IP address of your Pi. If you’re not sure how to find it, check your router’s web interface to see if it gives a list of clients or run a NMAP scan.

  10. It’s general a good practice to immediately check for and install updates to make sure the image does not have any well-known security issues:

    sudo apt-get update && sudo apt-get dist-upgrade`

You’ll want to take some other steps to configure and secure the Raspberry Pi, but I’ll save that for another post. The above should have you up and running.

References:

  • https://www.raspberrypi.org/documentation/installation/installing-images/
  • https://www.raspberrypi.org/downloads/raspbian/
