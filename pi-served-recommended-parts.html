Pi Served: Recommended Parts

If this is your first Raspberry Pi, you need to make sure you have the right parts before you can really start playing with it. My recommendations for a home Linux server are below, but you can of course use alternatives to your liking/budget (the last two are completely optional accessories).

  • Apple 12W USB Power Adapter - I like the Apple version because it’s rated for 5.2 volts and 2.4 amps which is more than sufficient to support overclocking and any Pi accessories I may decide to add in the future. Also, when it comes to electrical stuff I’d rather by a brand which should have some good quality control and safety measures in place.
  • Samsung Pro Plus 32GB MicroSDHC Memory Card with 95MB/s Read, 90MB/s Write - This card likely supports read/write speeds faster than the Pi is capable of reading/writing, but I got a great deal on at the time and don’t have to worry about the card being a bottleneck. However, at the current price at the time of this post, I suggest you also look at the SanDisk Extreme 16GB microSDHC UHS-1.
  • Inexpensive Ethernet Cable - I had a spare one in my parts bucket and any basic cable should do so I don’t have a specific recommendation. A wireless USB adapter is alternative option, but I prefer wired connections when practical.
  • Micro-USB Cable - As with the Cat5 cable, I had a spare so I don’t have a specific recommendation. If in doubt, the AmazonBasics brand should be a safe choice.
  • Inexpensive Heatsink Set - If you plan to overclock and keep this in an enclosed, fan-less case, a heatsink might help. I received some as a gift and am not sure of which brand I have. Any set with decent reviews online should meet your needs.
  • Premium Clear Case for Raspberry Pi 2 Model B Quad Core and Raspberry Pi Model B+ (B PLUS) - This is a matter of preference. I like the clear style and accessibility to all the ports and pins provided by this case.

If you’re new to the Pi like I was, I hope the above list will be a good starting point for acquiring the accessories needed for a home server project.
